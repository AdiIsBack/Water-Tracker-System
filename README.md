# 💧 Water Tracker

A simple web-based Water Tracker app built using HTML.  
It helps users track their daily water intake easily.

---

## 🚀 Features
- Add water intake entries
- View history of water consumption
- Simple and lightweight UI

---

## 📁 Files
- `index.html` → Main tracker page
- `history.html` → Shows water intake history

---

## 🛠️ Built With
- HTML
